git clone (traer el repositorio desde git al editor)
git ls (lista los archivos y carpetas de la ruta)
git cd (te mueve a la ruta seleccionada, con ../ se devuelve una ruta antes)
git status (muestra la condicion del los archivos. Si han sido modificados o no)
git add (añade el cambio de mi archivo para hacer el commit)
git add . (añade todos los cambios de todos los archivos para el commit)
git reset (quita un archivo de la zona de preparacion antes de haver el commit)
git commit -m "mensaje" (colocar mensaje significativo segun el cambio)
git push origin main (sube mis cambios de codigo a mi repositorio)
git pull origin main (baja los cambios del repositorio)
git branch (muestra mis ramas y en la que me encuentro con un * antes del nombre de la rama)
git checkout -b feature/nomre_branch (crear una rama)
git checkout "nombre de la rama" (para cambiar de rama)
git checkout "nombre de archivo" (para quitar cambios que no van en esa rama)
git branch -D nombre de la rama (para eliminar una rama)
